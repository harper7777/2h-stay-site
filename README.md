# 2H分享家民宿網站

以正式部署為目標整理的靜態前端專案。採用共用 header/footer、資料驅動房型卡片、模組化 CSS 與可擴充 JS 架構。

## 技術

- HTML5
- CSS3
- JavaScript
- Bootstrap Icons
- RWD
- Scroll Animation
- Data-driven room cards

## 執行方式

請使用 VS Code Live Server 或任一靜態伺服器開啟，避免瀏覽器以 file:// 開啟時無法 fetch partials。

## 結構

```text
partials/    共用 Header/Footer
css/         base/layout/page CSS
js/          layout、動畫、頁面邏輯
data/        房型資料
```
